# AI Visual Inspection System with Generative AI Reports

An end-to-end computer vision application that analyzes product images using deep learning and generates automated inspection reports using Generative AI.

## Features
- Image upload interface
- OpenCV preprocessing
- PyTorch ResNet inference
- Defect classification
- Automatic text report generation
- Flask web application

## Tech Stack
Python, OpenCV, PyTorch, Flask, Transformers

## How to Run
pip install -r requirements.txt

python app.py
